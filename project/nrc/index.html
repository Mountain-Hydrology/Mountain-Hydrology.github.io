Coupling snow sub-grid variability model into a land-surface model

Siwei He
Aug 1, 2018
More information about the NRC program

This project aims to solve the spatial variability of snow depth problem in the Earth system modeling. A snow model for representing sub-grid variability of snow depth will be coupled into the Rapid Update Cycle Land Surface Model (RUC LSM), and its impacts on RUC LSM will be evaluated.
